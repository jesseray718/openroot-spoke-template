# OpenRoot Spoke Template

Fork this to create a new OpenRoot spoke module.

## Structure
docs/  — design docs, build guides, photos
cad/   — design files (FreeCAD, OpenSCAD)
bom/   — bill of materials with sources + costs
src/   — control code, scripts, firmware

## Conventions
- License: CC-BY-SA-4.0 (docs) / GPL-3.0 (code)
- Tag maturity: Concept | Modeled | Prototyped | Field-tested
- Link your spoke in the [OpenRoot hub](https://github.com/jesseray718/openroot)
